PT Ding Sheng Wijaya Website - IT Handoff

This handoff package contains the latest bilingual (Indonesia / English) website version.

Files included:
- index.html
- site.css
- assets/dingsheng logo.jpg
- single-file.html

Recommended hosting method:
1. Upload the contents of this handoff folder to the web server document root or target domain folder.
2. Keep this structure intact:
   - index.html
   - site.css
   - assets/dingsheng logo.jpg
3. Ensure the web server serves index.html as the default homepage.
4. Confirm static files are publicly reachable.

Language support:
- The website includes a top-right language switcher: ID / EN.
- Default language is Indonesian.
- The selected language is saved in the visitor's browser.

Domain setup:
- Point the domain or subdomain to the hosting server provided by your IT infrastructure.
- Once DNS resolves, this website can be served directly from that domain.

Notes:
- index.html is the main homepage.
- site.css contains the styling.
- assets/dingsheng logo.jpg is required for the logo.
- single-file.html is a simplified backup version.

If your IT team prefers a single file deployment, they can use single-file.html and rename it to index.html.
